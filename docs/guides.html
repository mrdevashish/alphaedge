Guides

1) The Checklist (Scan → Filter → Define → Annotate → Review)

  1. Scan: large-caps & liquid mid-caps for trend + momentum alignment.
  2. Filter: volatility bands, structure (break/retest), volume context.
  3. Define: entry, SL, targets before the trade.
  4. Annotate: “strengthens/weakens” notes for confidence.
  5. Review: learn, don’t chase hindsight wins.

2) Risk (Position sizing)

Size by risk per trade (e.g., 0.5–1% of capital). Keep RR realistic (e.g., 1:1.3–1:2). Respect SL.

3) Journaling

Log entries, exits, RR, emotion, mistake types. Review weekly. Capture “what I’d repeat / avoid”.
